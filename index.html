About this page
  • The map reads from data/latest_obs.txt when hosted as a static site.
  • A scheduled workflow can refresh that file every hour without adding a backend service.
  • If you open this page with file://, use Import File or drag in a NOAA text file to load fresh observations.
  • Land and borders are bundled from Natural Earth 1:50m geometry so the basemap stays local.
